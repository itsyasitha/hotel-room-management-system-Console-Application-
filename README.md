# Hotel Room Management System (Terminal Application)

A simple C++ console application for managing hotel room reservations. This program allows users to:

* Check room availability
* Reserve rooms of different types
* Track rooms sold
* Calculate daily collection and sales information

## Features

* Supports Single, Double, Family, Deluxe, Suite, and Penthouse rooms
* Displays remaining rooms and total collection for each type
* Provides a summary of total sales at the end

## Usage

1. Compile the C++ program using a C++ compiler:

```bash
g++ hotel.cpp -o hotel
```

2. Run the executable:

```bash
./hotel    # On Linux/macOS
hotel.exe  # On Windows
```

3. Enter the number of available rooms for each category.
4. Use the menu to reserve rooms or view sales information.
5. Follow on-screen instructions to navigate through options.
